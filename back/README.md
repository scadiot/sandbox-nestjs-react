## Installation

```bash
# install app
$ yarn install

# Push Prisma schema on database
$ yarn prisma:push
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
